## Ferrite Nodes

- **Ferrite Dust (Generic)** -> `Toxo-gite`
- **Oxygen** -> `Toxo-xite`
- **Sodium** -> `Toxo-natrine`
- **Carbon** -> `Toxo-carbite`
- **Di-hydrogen** -> `Toxo-hydrite`
- **Chromatic Metals** -> `Toxo-chromite`
- **Tritium** -> `Toxo-trinite`
- **Copper** -> `Toxo-cuprite`
- **Silver** -> `Toxo-argenite`
- **Ionised Cobalt** -> `Toxo-cobaltite`
- **Salt** -> `Toxo-salite`
