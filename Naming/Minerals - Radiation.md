- **Ferrite Dust (Generic)** -> `Rado-gite`
- **Oxygen** -> `Rado-xite`
- **Sodium** -> `Rado-natrine`
- **Carbon** -> `Rado-carbite`
- **Di-hydrogen** -> `Rado-hydrite`
- **Chromatic Metals** -> `Rado-chromite`
- **Tritium** -> `Rado-trinite`
- **Copper** -> `Rado-cuprite`
- **Silver** -> `Rado-argenite`
- **Ionised Cobalt** -> `Rado-cobaltite`
- **Salt** -> `Rado-salite`
